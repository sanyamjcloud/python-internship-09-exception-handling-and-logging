# Python Error Handling and Logging

## Objective
This project demonstrates how to handle runtime errors in Python using:
- try-except blocks
- Multiple exception handling
- else and finally blocks
- Custom exceptions
- Logging errors to a file

## Files
- `error_handling.py` → Main Python program
- `error.log` → Automatically generated error logs
- `README.md` → Project explanation

## Concepts Used
- ZeroDivisionError
- ValueError
- Custom Exception
- Logging module
- Runtime error simulation
- Debugging using logs

```bash
python error_handling.py
